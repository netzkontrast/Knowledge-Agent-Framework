# Provenance — Reference Material Only

**WARNUNG — NICHT in den Langdock-Wissensordner hochladen.**

This folder contains the build-time research inputs used to author the
knowledge base:

- `sources/` — ingested source documents (platform feature inventories,
  marketing scenario catalogs, FAQ corpora, persona research).
- `research/` — deep-research reports and the advanced-scenario set.
- `extracts/` — the synthesized theme extracts (T0–T8) that fed the files.

These are included for **transparency and audit only**. They may contain
`[UNVERIFIED]` / `[INFERRED]` notes and statements that were superseded or
reconciled during authoring. They are NOT the deployed knowledge base.

**The authoritative, deployable knowledge is the `knowledge/` folder only.**
Uploading provenance files would reintroduce resolved contradictions and
dilute retrieval (Langdock returns one chunk per file per query).
